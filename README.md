## 프로젝트 목적

- 운동 기록을 통해 운동 시간과 운동 강도를 확인하고 조절할 수 있게 합니다.
- 꾸준히 운동하였음을 달력을 통해 확인하고 성취감을 얻을 수 있게 합니다.
- 자신이 만든 루틴을 다른 유저들과 공유할 수 있고, 다른 유저들의 루틴을 받아 수행할 수 있습니다.

<br>
